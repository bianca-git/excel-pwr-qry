How It Works: The Big Picture

This guide demonstrates a powerful, automated workflow in Excel. You will learn how to point Power Query to a folder of similarly structured PDFs, define a cleaning process for one file, and then have Excel automatically apply those same steps to every file in the folder. The result is a single, clean table ready for analysis, updated with one click.

The Automated Workflow

1. A Folder of PDFs

All your source files, in one place.

2. Power Query Magic

Teach Excel once how to clean and transform the data.

3. One Clean Excel Table

All your data, combined and ready for analysis.

What You'll Need

Excel for Microsoft 365

Power Query is a core, built-in feature in modern desktop versions of Excel.

Consistently Structured PDFs

The process works best when all your PDFs have the same layout and table structure.

A Dedicated Folder

Create a specific folder that will contain *only* the PDF files you want to combine.

The Core Process: A Guided Walkthrough

Follow these five steps to build your automated query. Click each tab on the left to see the detailed instructions and a corresponding screenshot for that part of the process. This is where you tell Excel how to do the heavy lifting.

Step 1: Connect to the Source Folder

The first move is to tell Excel where to find your files. This establishes the connection to the folder, allowing Power Query to access its contents.

  1. Begin with a new, blank Excel workbook.
  2. Go to the Data tab on the ribbon.
  3. Click Get Data > From File > From Folder.
  4. Browse to and select your folder, then click Open.
  5. In the preview window, click the dropdown on the Combine button and select Combine & Transform Data. This opens the Power Query Editor.

Step 2: Define the Transformation

You'll now teach Power Query how to clean the data by working with a single sample file. Every action you take here is recorded and applied to all other files.

  1. In the "Combine Files" window, select the table or page from the sample PDF that contains your data.
  2. Click OK to enter the main Power Query Editor.
  3. Use the tools in the ribbon to clean the sample data:
    • Use "Use First Row as Headers".
    • Change data types.
    • Remove unnecessary columns.

Step 3: Review Combined Data

Power Query automatically applies your steps to all files and stacks them into one table. Review the result.

  1. Scroll through the data to check for correctness.
  2. The Source.Name column shows the origin filename for each row, which is useful for verification.
  3. If you see errors, you can go back to the "Transform Sample File" query to adjust your steps.

Step 4: Final Polish

Perform final checks on the combined dataset before loading it into Excel.

  1. Double-check that all columns have the correct data type (e.g., number, text, date).
  2. Rename or remove the Source.Name column if you don't need it.
  3. In the "Query Settings" pane on the right, give your query a descriptive name (e.g., `CombinedPDFData`).

Step 5: Close & Load to Excel

The transformation is complete. It's time to send the clean, combined data back to your Excel worksheet.

  1. On the Home tab of the Power Query Editor, click the Close & Load button.
  2. Power Query creates a new worksheet and loads your data into a formatted Excel Table.
  3. Your data is now ready for analysis, PivotTables, or charts.

Key Techniques & The Payoff

These are the most common actions you'll take to clean your data in Power Query, plus an explanation of how to refresh your query when new data arrives.

Common Data Cleaning Steps

Use First Row as Headers

Promotes the first row of your data to become the column headers. Usually the first step after connecting.

Change Data Type

Essential for calculations and sorting. Set columns to Text, Whole Number, Decimal, Date, etc.

Remove & Reorder Columns

Right-click a column header to remove it. Drag and drop headers to change their order.

Filter Rows

Use the filter dropdown on any column to exclude rows you don't need, such as totals, subtotals, or blanks.

The Payoff: One-Click Refresh

The initial setup is a one-time investment. The real power is in the refresh. When you add new PDFs to your source folder (or remove old ones), simply go to the Data tab in the main Excel window and click the Refresh All button. Power Query will instantly re-run all your defined steps on the folder's new contents and update the final Excel table. This transforms a repetitive, manual task into a fully automated, one-click process.
