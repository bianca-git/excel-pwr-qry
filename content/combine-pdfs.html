Combining PDFs: The Automated Workflow

Learn how to point Power Query to a folder of PDFs, define a cleaning process once, and have Excel apply it to every file automatically. The result is a single, clean table, ready for analysis and updated with one click.

1

Connect to the Source Folder

First, tell Excel where to find your files. This establishes the connection to the folder, allowing Power Query to access its contents.

  • Begin with a new, blank Excel workbook.
  • Go to the Data tab on the ribbon.
  • Click Get Data > From File > From Folder.
  • Browse to and select your folder, then click Open.
  • In the preview window, click the dropdown on the Combine button and select Combine & Transform Data.

2

Define the Transformation

Teach Power Query how to clean the data by working with a single sample file. Every action you take here is recorded and applied to all other files.

  • In the "Combine Files" window, select the table or page from the sample PDF that contains your data.
  • Click OK to enter the main Power Query Editor.
  • Use the tools in the ribbon to clean the sample data, such as "Use First Row as Headers," changing data types, and removing unnecessary columns.

3

Review Combined Data

Power Query automatically applies your steps to all files and stacks them into one table. Review the result.

  • Scroll through the data to check for correctness.
  • The Source.Name column shows the origin filename for each row, which is useful for verification.
  • If you see errors, you can go back to the "Transform Sample File" query to adjust your steps.

4

Final Polish

Perform final checks on the combined dataset before loading it into Excel.

  • Double-check that all columns have the correct data type (e.g., number, text, date).
  • Rename or remove the Source.Name column if you don't need it.
  • In the "Query Settings" pane on the right, give your query a descriptive name (e.g., `CombinedPDFData`).

5

Close & Load to Excel

The transformation is complete. It's time to send the clean, combined data back to your Excel worksheet.

  • On the Home tab of the Power Query Editor, click the Close & Load button.
  • Power Query creates a new worksheet and loads your data into a formatted Excel Table.
  • Your data is now ready for analysis, PivotTables, or charts.

6

Refresh Your Data (The Best Part)

You've built a connection, not just a one-time import. If the source data changes, you don't have to repeat these steps. Just refresh!

To refresh the data:

  • Go to the Data tab and click the Refresh All button.
  • Alternatively, right-click anywhere inside the new data table and select Refresh.

Excel will automatically re-run all your saved steps and pull in the latest data from the source file. No more manual updates.
